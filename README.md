## Что это?
Телеграм-бот системы платежей, написанный с использованием aiogram 3
## Какие у него функции?
- переводы другим людям
- просмотр таблиц лидеров по валюте (SCC)
- создание уникальных товаров, которые можно оплатить более быстрым путём, чем при обычном переводе

Этот проект больше не разрабатывается
